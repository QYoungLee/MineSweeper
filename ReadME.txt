이 게임은 시각장애인들도 할수 있도록 타일과 게임 오버 여부를 소리로 알려주는 지뢰찾기 게임입니다.

1. 방향키로 커서를 움직입니다. 커서를 움직일 때마다 맨 왼쪽 위 타일을 (1,1)로 규정하고 소리로 현재 커서가 있는 타일을 좌표로 불러줍니다.
2. 커서의 위치가 있는 곳의 상태를 확인하기 위해 자판 s를 누릅니다. 그러면 타일 s의 상태를 확인할 수 있습니다. s를 중심으로 q,w,e,a,d,z,x,c를 누르면 대응되는 방향의 타일의 상태를 확인할 수 있습니다. 이를태면 a를 누르면 커서 바로 왼쪽에 있는 타일의 상태를 소리로 확인할 수 있습니다.
3.스페이스바를 누르면 해당 타일을 청소하게 됩니다. 지뢰가 있는 타일이라면 게임 오버 됩니다.
4. v를 누르면 커서가 위치한 타일에 깃발을 꽂을 수 있게 됩니다. 모든 지뢰 위치에 깃발을 꽂고 다른 타일을 모두 닦으면 게임 클리어입니다. 깃발이 있는 타일은 닦을수 없게 비활성화 되고 v를 한번 더 누르면 다시 활성화 되어 닦을 수 있게 됩니다.
5. 게임오버될 경우 esc버튼을 통해 게임을 처음부터 다시 시작할 수 있습니다.
